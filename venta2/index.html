METODOLOGIAS DE DESARROLLO DE SOFTWARE

Enfoques estructurados que guían la planificación, implementación y mantenimiento de sistemas de software, asegurando que los requisitos sean definidos, comprendidos y gestionados adecuadamente

Cada metodología influye en la forma en que se define el alcance de los requerimientos:

1. Cascada:

Define requisitos detallados desde el inicio y sigue un proceso secuencial.

2. Ágil (Scrum, Kanban, XP):

Define requisitos de forma iterativa e incremental, adaptándose a cambios.

3. Modelo en V:

Similar a Cascada, pero con pruebas en cada fase.

4. DevOps:

Integra desarrollo y operaciones, permitiendo una entrega continua.
